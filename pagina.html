17. Script Python - Calcular Factorial

Este script calcula el factorial de un número.


def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

print(factorial(5))  # Output: 120

21. Script Python - Comprobar si un Número es Primo

Este script verifica si un número dado es primo.


def es_primo(num):
    if num < 2:
        return False
    for i in range(2, int(num**0.5) + 1):
        if num % i == 0:
            return False
    return True

print(es_primo(7))  # Output: True
